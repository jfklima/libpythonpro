# libpythonpro
Módulo para exemplificar construção de projetos Python no curso PyTools

Nesse curso é ensinado como contribuir com projetos de código abarto.

Link para o curso [Python Pro](https://www.python.pro.br/)

[![Build Status](https://app.travis-ci.com/jfklima/libpythonpro.svg?branch=main)](https://app.travis-ci.com/jfklima/libpythonpro)

Suportada a versão 3 do Python.

Para instalar:

```console
python3 -m venv .venv
source .venv/bin/activate
pin install -r requirementis-dev.txt
```

Para conferir qualidade no código:

```console
flake8
```
